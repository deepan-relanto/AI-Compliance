# Enterprise AI Fundamentals · Interactive Mind Map

Standalone Relanto-branded mind map built from the five NotebookLM screenshots.

## Open it

Open `mindmap-01.html` in Chrome/Edge (double-click).

## What’s inside

One combined tree under **Enterprise AI Fundamentals**:

1. **Core Concepts** — AI, ML, GenAI, LLMs, Prompt, Context  
2. **Key Terminology** — Hallucinations, Tokens, Training Data  
3. **Automation vs AI** — Traditional automation vs AI  
4. **Workplace Applications** — HR & Admin, Sales & Marketing, Finance & Planning, Project Management  
5. **Limitations & Risks** — Accuracy, Knowledge, Responsibility  
6. **Best Practices** — Prompting, Verification, Security & Usage, Efficiency  

This lines up with the learning themes in the repo’s `questions.json`.
